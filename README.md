# HMS Odpočet

Utilita na odpočet času a vypnutí PC

## Funkce

- odpočet času
- vypnutí PC nebo odhlášení uživatele
- zobrazení miniaturní aplikace vždy v popředí
- znázornění odpočtu progress barem na hlavním panelu
- minimalizace do tray ikony
- systémová oznámení
